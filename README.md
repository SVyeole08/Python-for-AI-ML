## 1. Python Learning Notes

**Files:** `main.py`, `python.md`

**Description:** Core Python fundamentals collected in one place — commented-out exercises in `main.py` and a structured guide in `python.md`.

**Topics covered:**

- String manipulation & slicing
- Type conversion (`int`, `float`, `str`, `tuple`)
- Input/output & f-strings
- Conditional statements (comparisons, even/odd, leap year, temperature, voter eligibility)
- `for` loops (multiplication table, sum, factorial, perfect/prime numbers, palindrome)
- `while` loops (digit extraction, number reversal, palindrome check)
- Data structures — List, Tuple, Set, Dictionary (methods, operations, traversal)
- Exception handling (`try` / `except` / `else` / `finally`, `raise`)
- File handling basics (read, write, append modes)

**How to run:**

```bash
python main.py
```

**Documentation:** See [`python.md`](python.md) for the full learning guide.

## 2. Number Guessing Game

**File:** `Numberguess.py`

**Description:** A console game where the computer picks a random number between 1 and 100, and the player guesses until they get it right. Gives "higher" / "lower" hints and counts total tries.

**Concepts used:** `random`, `while` loop, conditionals, user input

**How to run:**

```bash
python Numberguess.py
```

## 3. File Handling CRUD App

**Folder:** `File Handling Project/`

**Files:**

- `app.py` — menu-driven CRUD application
- `fileHandling.md` — project documentation

**Description:** A command-line file manager that performs Create, Read, Update, and Delete operations on files using Python's `pathlib` module.

**Features:**

- Create new files with custom content
- Read and display file content
- Update files — rename, append, or overwrite
- Delete files
- Error handling and file-existence checks

**How to run:**

```bash
cd "File Handling Project"
python app.py
```

**Documentation:** See [`File Handling Project/fileHandling.md`](File%20Handling%20Project/fileHandling.md).
